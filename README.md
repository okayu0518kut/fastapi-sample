# fastAPIとbootstrap5でつくった電話帳あぷり
- docker-composeでアプリコンテナとnginxコンテナを管理する
### Requirements
- Docker
### Getting Started
```
docker compose up
```
### 技術スタック:
- フロントエンド: BootstrapでスタイリングしたHTML/CSS/JavaScript
- バックエンド: FastAPI
- データベース: SQLite
- Docker: 開発環境の統一、バックエンド・データベースのコンテナ化
- GitHub: コード管理、ブランチ戦略、プルリクエストの実践
